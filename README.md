# ProColor
Advanced WireColor Manager for 3ds Max



## Install
Unzip
Drag & drop in a viewport
 
## Features

Random Gray Wirecolor
Random Color Wirecolor
Gradient Wirecolorr
Realtime Solid Wirecolor
Material Eraser
